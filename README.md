# War Game Project

This project is a war game application developed using the Java programming language.

## Overview
The game includes knights and enemies. Knights are assigned default good and bad luck attributes, and the user is asked to make selections based on these attributes. After defining the types of active knights, the game starts based on the player's performance. At the end of the game, the results and scores are displayed on the screen.

## Features
- Knight and enemy characters
- Luck-based character attributes
- User interaction and selection
- Game start and score calculation
- Result display at the end of the game

## Technologies
- Java
- Object-Oriented Programming (OOP)
